# XAU 三周期交易系统 · 一页式操作清单

> 来源：HoneyRay 分享 | 2026-03-24

## 核心框架

| 周期 | 职责 | 禁止 |
|------|------|------|
| **H1** | 只定方向（偏多/偏空/震荡） | 不开仓，不来回改主意 |
| **M15** | 主战场，只做 Zone① + 结构确认 | 不在 H1 开仓 |
| **M3** | 只抠价格，等回调到位 | 不追单，不自己找信号 |

## H1 判断规则

**只看4个东西：**
1. HH/HL/LH/LL 序列 → HH+HL=偏多；LH+LL=偏空；乱=震荡
2. CHoCH → 刚出空头 CHoCH，警惕多转空
3. 威科夫 H1 面板 → 只看行0、行3
4. EMA200 → 上方偏强，下方偏弱

**输出只有3个：**
- 偏多 → 今天 M15 只找多
- 偏空 → 今天 M15 只找空
- 震荡 → 只做上下沿，不追中间

**停手条件（满足任意2条先不做）：**
- 刚出 CHoCH
- 威科夫显示"等待/禁手/没油了"
- 价格贴着大级别关键区

## M15 入场规则（4条必须齐）
1. Zone① 首次触及
2. 方向和 H1 一致
3. 有 CHoCH / FVG / Sweep 确认
4. 威科夫行0 不是禁手

**加分项：** 附近有 EQH/EQL 被扫过；Zone 与 PDH/PDL 共振

## M3 规则
- 等价格回到 M15 行1 提示的位置附近
- 到了看：小结构确认 + 回踩 EMA21 是否顺畅 + 止损能否放有效高低点外侧
- 到了目标价 → 可以进
- 没到目标价 → 继续等
- 直接冲走 → 不追
- 到了但结构没确认 → 不进

**仓位判断：**
- 波段回收 <70% → 正常仓位
- 70%-85% → 轻仓
- >85% → 不进
- 反方向 Zone 保护距离不够 → 放弃

## 威科夫用法
| 行0 | 行动 |
|-----|------|
| 禁手 | 不做 |
| 等待 | 继续等 |
| 可做多/可做空 | 进入下一步 |

| 行1 | 行动 |
|-----|------|
| 给目标价 | 等回调，不追 |

## 下单前 6 句自检
1. H1 现在是不是偏多/偏空清楚？
2. M15 现在是不是 Zone① ？
3. 有没有 CHoCH / FVG / Sweep 确认？
4. 威科夫是不是没亮禁手？
5. M15 行1 的回调目标价，我是不是等到了？
6. 反方向 Zone 距离够不够？

**规则：6句里有1句答不上来，就不开。**

## 极简口令

- M15：只做 Zone①
- M15：红灯亮了停手
- M3：只等回调到位
- M3：不追
- 进了交给 EA
