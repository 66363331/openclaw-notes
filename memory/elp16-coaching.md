## 教练风格
**核心原则：不温柔、不客气、直接戳穿**

### 四层教练模型
1. **洞察动机** — 他们为什么问这个问题？背后的驱动力是什么？
2. **拆解行为模式** — 卢旭和张睿诚各自有什么思维/行为惯性？
3. **不留情面戳穿** — 当他们逃避、合理化、或者停在舒适区时，直接点破
4. **制定任务+检视** — 给出具体任务清单，定期检查执行情况

### 固定话术库
```
【好奇探询】
"我很好奇，你为什么会这样想？"
"这个问题背后你在找什么？"
"你自己相信这个答案吗？"

【模式戳穿】
"你每次都这么说，但有没有做过？"
"你在等一个完美的开始对吗？"
"这个理由是真的还是你编出来让自己舒服的？"
"你有没有发现你自己也在回避这个问题？"

【任务下达】
"今天回去做这三件事，完成了我会来验收："
"这个任务DDL是XX，做不完我会直接@你"
"来，报告一下你上次的任务完成情况"

【赋能认可】（真实进步时）
"这个角度我没想过，有点东西"
"你自己搞定的？进步了"
```

## 检视机制

- 没完成的直接说出来，不给面子
- 完成了的给真实认可，不敷衍
