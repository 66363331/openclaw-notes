# 正确的三层周期进场方案

## 📊 周期架构

```
┌─────────────────────────────────────────────┐
│  H1 (1小时) - 趋势判断层                      │
│  └── EMA20/50/200 排列判断方向               │
│  └── 只开顺势单（多头排列只做多）            │
├─────────────────────────────────────────────┤
│  M15 (15分钟) - 位置确定层                    │
│  └── SMC结构检测（Swing High/Low）           │
│  └── CHoCH突破确认                           │
│  └── 生成OB订单块区域                        │
├─────────────────────────────────────────────┤
│  M3 (3分钟) - 实际进场层 ⭐                   │
│  └── 价格回到M15生成的OB区域                 │
│  └── K线形态确认（阳线/吞没/Pinbar）         │
│  └── 出现绿色箭头 = 开仓                     │
└─────────────────────────────────────────────┘
```

---

## 🎯 进场流程

### Step 1: H1趋势确认
```python
if 价格 > EMA20 > EMA50 > EMA200:
    趋势 = "牛市" → 只寻找做多信号
elif 价格 < EMA20 < EMA50 < EMA200:
    趋势 = "熊市" → 只寻找做空信号
else:
    趋势 = "震荡" → 不交易
```

### Step 2: M15结构确认
```python
# 检测CHoCH突破
if 价格突破前高(Swing High):
    标记Bullish结构
    创建Bull OB（突破前那根K线的高低区域）
    
if 价格突破前低(Swing Low):
    标记Bearish结构
    创建Bear OB
```

### Step 3: M3实际进场 ⭐
```python
# 只有在H1趋势方向一致时才进场
if H1趋势 == "牛市" 且 有Bull OB:
    if M3价格触及OB上沿 且 M3收阳线:
        做多进场！
        
if H1趋势 == "熊市" 且 有Bear OB:
    if M3价格触及OB下沿 且 M3收阴线:
        做空进场！
```

---

## 💡 关键区别

| 错误理解 | 正确理解 |
|---------|---------|
| H1直接进场 | H1只判断方向，不开仓 |
| M15直接进场 | M15确定OB区域，不开仓 |
| M3只看均线 | M3看OB+K线形态，**这才是进场点** |

## ⚠️ 重要提示

**数据限制：**
- 当前只有2025年3-4月（2个月）tick数据
- 完整回测需要更多历史数据（至少6-12个月）
