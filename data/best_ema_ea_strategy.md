# EMA进场 + EA管理 最优参数方案

## 📊 进场逻辑设计

### 趋势判断 (H1周期)
```
多头排列：价格 > EMA20 > EMA50 > EMA200 → 只做多
空头排列：价格 < EMA20 < EMA50 < EMA200 → 只做空
```

### 进场时机 (M15周期)
```
多头：价格回调触及EMA20后反弹做多
空头：价格反弹触及EMA20后回落做空
```

## 📋 使用建议

### 1. 趋势判断 + 参数选择
```
强势单边趋势 → 用方案A (回撤30%，大止盈)
震荡盘整行情 → 用方案B (回撤20%，快止盈)
不明朗趋势 → 用方案C (平衡参数)
```

### 2. 开仓过滤条件
```python
# 建议增加的开仓过滤
1. H1趋势明确 (EMA排列清晰)
2. M15回调到位 (触及EMA20)
3. K线形态确认 (Pinbar/吞没)
4. 避开数据公布前后30分钟
5. 日内最多1-2笔交易
```

### 3. 风险控制
```
单日最大亏损: -$500 (达到停止交易)
连续亏损3次: 暂停开仓，检查趋势
硬止损必须设置: 不可取消
```
