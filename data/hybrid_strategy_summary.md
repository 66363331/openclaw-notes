# EMA趋势过滤 + SMC结构进场 + EA管理 综合方案

## 📊 方案设计

### 1. 趋势过滤 (EMA)
```
H1周期判断：
- 多头排列 (价格>EMA20>EMA50>EMA200) → 只允许做多
- 空头排列 (价格<EMA20<EMA50<EMA200) → 只允许做空
- 无排列 → 不交易
```

### 2. 结构进场 (SMC)
```
M15周期执行：
- CHoCH结构突破后生成OB订单块
- 价格回到OB区域
- K线形态确认 (吞没/Pinbar/阳线)
- 开仓
```

### 3. EA管理
```
- 金字塔加仓 (顺势浮盈时)
- 马丁补仓 (逆势回调时)
- 追踪止盈/主止盈
- 硬止损保护
```

## 📈 预期表现 (基于数据推算)

| 指标 | EMA过滤 | SMC进场 | 综合 |
|------|---------|---------|------|
| 交易次数 | ~50笔/2月 | ~10笔/2月 | ~15-20笔/2月 |
| 胜率 | 65-70% | 50-60% | 60-65% |
| 盈利因子 | 1.8-2.2 | 1.2-1.5 | 1.6-2.0 |
| 最大连亏 | 3-4次 | 2-3次 | 2-3次 |

**优势：**
- EMA过滤减少逆势交易
- SMC提供更精准的进场点
- 综合后胜率提高，连亏减少

## ⚠️ 风险提示

1. **硬止损仍是最大风险** - 单笔-$220可能吃掉多笔利润
2. **趋势转折期表现差** - EMA排列失效时连续亏损
3. **数据量不足** - 2个月数据不足以验证长期表现

**建议：**
- 先用小资金测试1-2个月
- 收集更多历史数据验证
- 考虑加入更宽止损或移动止损
