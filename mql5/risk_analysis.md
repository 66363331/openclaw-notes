- **手数**: 0.01

## ⚠️ 风险管理分析

### 1. 仓位大小计算
**假设账户资金**: $10,000
**风险比例**: 1% per trade

```
单笔最大亏损 = $10,000 × 1% = $100
止损点数 = 50点
每点价值 = $1 (XAU/USD 0.01 lot)

理论手数 = $100 ÷ $50 = 0.02 lot

✅ 当前设置 0.01 lot = 0.5% 风险 (更安全)
✅ 可承受连续 20 次亏损
```

### 2. 回撤分析

| 连续亏损次数 | 累计亏损 | 账户剩余 | 回撤比例 |
|-------------|---------|---------|---------|
| 3次 | $150 | $9,850 | 1.5% |
| 5次 | $250 | $9,750 | 2.5% |
| 10次 | $500 | $9,500 | 5.0% ⚠️ |

**结论**: 连续10次亏损后回撤仅5%，风控良好 ✅

### 3. 盈亏比分析

```
胜率要求计算:
盈亏比 = 1:2

盈亏平衡胜率 = 1 / (1 + 2) = 33.3%

目标胜率: 50%+
安全边际: 50% - 33.3% = 16.7% ✅ (充足)
```

### 4. 每日风险限制

```
MaxDailyTrades = 5
每单风险 = $50
每日最大风险 = $250 (2.5% 账户)

✅ 合理范围
```
